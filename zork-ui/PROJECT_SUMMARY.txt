PROJECT OVERVIEW
================================================================================
A production-ready terminal user interface for playing Zork (interactive 
fiction) through a Python ZIL (Zork Implementation Language) interpreter.

Built with Bun runtime and TypeScript for modern, fast, type-safe execution.
